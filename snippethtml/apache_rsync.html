Google around, and found the solution in SO.

You just need to modify all your virtual hosts file to new extension in /etc/apache2/site-enabled

For example:

 Original virtual file is mywebsite

        Change to mywebsite.conf

        Also, you need to create new symlink in /etc/apache2/site-available

rsync  stupid syntax

 if you want to rsync one file type[e.g.  *.html], then you can use

 rsync   -rvz  --include='*.html'  --exclude  “.”  source  dest

 

 If you want to rsync more than one file type[e.g. *.html and *.jpg] then you need sth different

 rsync   -rvz  --include='*'{html,jgp}  --exclude  “.”  source  dest
