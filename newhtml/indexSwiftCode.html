Swift is Apple new programming language for iOS, it it much better than Objective-C from many aspects 
        1. Functional style
        2. Use optional
        3. Take many "good" features from C/C++/Java/C# and Haskell
Something that I love in Swift 
        0. Mutable and Immutable variable with var and let
           var num = 3
           num can changed or reuse
           e.g. you can do: 
           num += 1  

           let x = 3
           x can NOT be changed or reuse
           x += 1 // get error âˆµ x is immutable variable

        1. Swift make initialize array very easy (if you want to initialize array in C++/Java/C#, it is very painful)
            var array = [Int]()             // create one dimensional array
            let array = [1, 2, 3]           // initialize one dimensional array with 1, 2, 3
            let array = [[1, 2], [3, 4]]    // initialize two dimentional array with [1, 2] , [3, 4]
