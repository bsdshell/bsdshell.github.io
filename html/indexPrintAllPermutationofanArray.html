$\langle {\large{\color{red}\alpha}} \rangle$ Print all the permutation of an array
This is very popular question for software engineer technical interview.

$\langle {\large{\color{red}\zeta}} \rangle$ This is very easy question to understand, but it is not so easy to code it up in 45 minutes. Let's try to understand that question.

Given array, char[] array = {'a', 'b', 'c'}, the following are all the permutations
a b c
a c b
c b a
c a b
b a c
b c a

The question is how we algorithmically print out the permutation?

1. If we fix the letter 'a', and permutate['b', 'c']
here is what we got?
'a', ['b', 'c']
'a', ['c', 'b']

2. we can swap 'a' to each char in ['b', 'c'] and ['c', 'b']
swap['a', ['b', 'c']] $\Rightarrow$ 'b', ['a', 'c'] and 'c', ['b', 'a']
swap['a', ['c', 'b']] $\Rightarrow$ 'c', ['a', 'b'] and 'b', ['c', 'a']

[a b c] $\Rightarrow$ [b a c] and [c b a]
[a c b] $\Rightarrow$ [c a b] and [b c a]

all the permutations[including the original array, [a b c] and [a c d]]
[a b c]
[b a c]
[c b a]
[a c b]
[c a b]
[b c a]
