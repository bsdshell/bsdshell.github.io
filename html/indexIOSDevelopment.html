$\square$ This is my first ObjectiveC tutorial
1. How to write the first Hello World in ObjectiveC
2. How to compile it in command line(yes, not in Xcode)

$\square$ The name of function is called main
The return type is int
One parameter is int argc
Other one is const char* argv[]

Each ObjectiveC has to include one and only one main function

@autoreleasepool is used to memory management

NSLog(@"Hello World") is just a function that print out message

In your command line, type
clang -fobjc-arc -framework Foundation HelloWorld.m

$\square$ clang is C/C++/ObjectiveC compiler
Foundation is defined a base layer of ObjectiveC classes $\square$
