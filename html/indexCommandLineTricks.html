$\langle {\large{\color{red}\alpha}} \rangle$ Change default location of Mac OSX screenshots, Open your Terminal and type: defaults write com.apple.screencapture location ~/myScreenShots
And type:
killall SystemUIServer

$\langle {\large{\color{red}\lambda}} \rangle$ Rename file name of default screenshots in Mac OSX, Open your Terminal and type:
defaults write com.apple.screencapture name "myName"
And type:

killall SystemUIServer

$\langle {\large{\color{red}\epsilon}} \rangle$ Print method name in Xcode/iOS:
NSLog(@"%s", __PRETTY_FUNCTION__);

$\langle {\large{\color{red}\sigma}} \rangle$ Compile Vim with Python support so that you can use clang_complete plugin for Clang family languages such as C/C++/ObjectiveC
I clone vim from github and compile it, but I have hard time to compile it.
One simple solution is to use macport, here is the command
sudo port install vim +python27

$\langle {\large{\color{red}\pi}} \rangle$ To find the clang on your Mac
xcrun -find clang

your output:
/Applications/Xcode.app/Contents/Developer/Toolchains/XcodeDefault.xctoolchain/usr/bin/clang

$\langle {\large{\color{red}\alpha}} \rangle$ locate libclang.dylib

$\langle {\large{\color{red}\zeta}} \rangle$ you need to set the path in your .vimrc file:
let g:clang_library_path='/Applications/Xcode.app/Contents/Developer/Toolchains/XcodeDefault.xctoolchain/usr/lib'
