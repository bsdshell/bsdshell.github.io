    • ------------------------------------------------------------------------------- Find root for any function

    Partition the interval [x0, xn] into list = [x0, x1) [x1, x2) [x2, x3) ..[x(n-1), xn) Cocat [xn, xn] with the list since the left boundary is checked only | Note: f(xn) might be zero, we need to check the xn boundary | TODO: need to check f(xn), DONE, concat [xn, xn] to the list | -------------------------------------------------------------------------------- | Good test cases: | f(x) = x^2 - 4, [-2, 2] | f(x) = x^5 -4*x^4 + 0.1*x^3 + 4*x^2 - 0.5, [-4, 4] => 5 solutions | -------------------------------------------------------------------------------- | limitation: | if each subinterval contains two or more values, then ONLY one value can be found | subinterval can be adjusted in arbitrary small | -------------------------------------------------------------------------------- | 0, 2/2 = [0, 1, 2] = [0, 1)[1, 2)
